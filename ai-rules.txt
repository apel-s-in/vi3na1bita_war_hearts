АРХИТЕКТУРА:

- vi3na1bita_war_hearts — отдельная игра для Game Center vi3na1bita-games.
- Публичный путь после деплоя: /Games/war_hearts/.
- Не создавать папку Games/ внутри этого репозитория.
- Корень репозитория деплоится в Yandex Object Storage prefix /Games/war_hearts/.
- Общий сетевой модуль подключается из /Games/common/network-bridge.js.
- Игра не должна читать OAuth token, parent localStorage, parent IndexedDB или parent.window.
- Связь с Game Center допускается только через postMessage или безопасные параметры запуска.
- Игра может иметь собственный WebRTC DataChannel, игровой чат и голосовой PTT внутри игровой сессии.
- Голос включается только по пользовательскому действию.
- На раннем этапе не добавлять service worker.
- Основная ориентация интерфейса: мобильная вертикальная 9:16.
- Игра должна работать standalone-preview, даже если Game Center или network bridge недоступны.
- Любые таймеры и polling должны останавливаться или деградировать при document.hidden.
